def start() выводит все виды услуг

def nav_cal_handler() выводит календарь для выбора даты

def process_simple_calendar() обрабатывает полученную дату и выводит времена для записей

def choose_time() обрабатывает полученое время и спрашивает адрес

def check_time() сравнивает полученную дату со списком занятых дат на день из Гугл Календаря

def address() принимает адрес от пользователя и спрашивает номер телефона

def phone() получает номер телефона и сохраняет все полученные выше данные в битрикс и гугл календарь

def register_handlers() регистрирует все хендлеры для вызова функций
