В файле requirements.txt написаны все библиотеки, которые используются в боте. Основные из библиотек:
- aiogram~=2.12 (Библиотека для создания бота)
- aiogram_calendar (Для выведения календаря в чат)
- python-dotenv==1.0.0 (Для скрытия токена бота)
- requests (Для запросов в Битрикс)
- и некоторые google библиотеки (Для запросов в Гугл Календарь)

остальные библиотеки установились автоматически. 

В файл env. нужно вставить токен своего бота. 

А еще я рекомендую сделать меню своего бота в Bot Father - ссылка на источник(https://yandex.ru/video/preview/3849028707349772925).

В файле BitrixAPI нужно вставить входящие веб хуки из вашего Битрикса ,а также вставить свои названия для созданных строк лида.
Чтобы найти название строки лида, нужно установить Приложение Поля в Битриксе. 

В файле GoogleCalendar нужно вставить свой скачанный json файл с ключами доступа от гугл Календаря.(Сейчас в проекте есть файл, но он пустой)
Для начала работы с Гугл календарем нужно активировать календарь функцией add_calendar(Она уже прописана в файле GoogleCalendar, ее нужно вызвать один раз)
